Classes

The classes for applications are worth 9 points. The classes for design are worth 6 points.

Applications

The 9 points are given for solving 9 problems, 5 in Haskell and 4 in Rust. You will receive an invitation by email to a test on HackerRank. (This is a platform used by many companies to filter candidates for software engineering positions.) You should receive the invitation for the Haskell test in Week 15, and the invitation for the Rust test in Week 17. If you did not receive the invitation, please let us know.

During classes, you can ask for help with the following tutorials:

  • Learning Haskell
  • Rust by Example or The Rust Book (your choice)

Your are probably ready for the Haskell test once you went through the first 3 sections of the Haskell tutorial. You are probably ready for the Rust test once you went through the first 9 sections of Rust by Example or through the first 6 sections in The Rust Book.
